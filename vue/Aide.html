Aide

  • Pour commencer l'enregistrement, appuyer sur le bouton puis réappuyer encore sur pour terminer.


  • Pour écouter l'enregistrement, appuyer sur le bouton puis réappuyer sur pour arreter.


  • Appuyer sur le bouton pour passer à un autre texte à lire.


  • Veuillez vous assurez de l'installation du plugin d' Adobe Flash dans votre navigateur pour éviter d'éventuelles messages d'erreur : https://get.adobe.com/fr/flashplayer/otherversions/
